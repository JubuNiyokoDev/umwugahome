# **App Name**: UmwugaHome Digital

## Core Features:

- Home Page with Chatbot: Landing page with key stats placeholders and a Gemini-powered chatbot for user orientation, suggesting relevant crafts and artisans. This chatbot acts as a tool to determine how best to serve user needs, as it makes decisions based on a model.
- Artisan Profiles: Verified artisan profiles with photos, bios, skills, and location; integrated with a simple shop for product display and basic order management.
- Training Center Profiles: Profiles for training centers to showcase programs, accept online enrollments. You will be able to make use of features like listing programs that the institution offers, as well as specifying prerequisites or necessary previous experience to make use of the courses offered
- Youth/Student Profiles: Basic profiles for students and recent graduates to connect with mentors.
- Marketplace: Searchable directory of artisans and centers; product showcase with simulated payments via placeholders.
- Training Space: Directory that allows navigation between courses and trainers.
- Admin Dashboard: Simple dashboard to manage accounts and content, with mock analytics data.

## Style Guidelines:

- Primary color: Gold (#FFD700) to evoke the richness of Burundian crafts, suitable for calls to action.
- Background color: Light beige (#F5F5DC) for a natural, earthy feel.
- Accent color: Burnt orange (#CC6633) analogous to gold, this warm, desaturated tone is suitable for highlights and secondary actions.
- Headline font: 'Belleza' (sans-serif) is recommended for headlines, aligning with fashion, art, and design.
- Body font: 'Alegreya' (serif) is recommended for the body, offering an elegant, intellectual, and contemporary feel for the application.
- Artisan-inspired icons with a modern, minimalist aesthetic.
- Multi-page layout for clear structure, responsive design, and server-side rendering.
- Subtle CSS or GSAP animations for page transitions, hover effects, and loading indicators to add a dynamic feel.
